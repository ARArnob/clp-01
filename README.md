## Description
This repository contains a single Python script that solves three specific problems as per the instructions:

1.  **Problem 1:** Finds the maximum of three integers using `if-else` logic (without `max()` or sorting).
2.  **Problem 2:** Calculates the digit count and digit sum of an integer (handles negative numbers).
3.  **Problem 3:** Sorts a list using Bubble Sort and searches for a target using Binary Search.

## How to Run

2.  Run the script in your terminal:
